HƯỚNG DẪN SỬ DỤNG CHƯƠNG TRÌNH HỢP GIẢI MỆNH ĐỀ


Cây thư mục của thư mục SRC như sau:
|---Folder input: 
	|--- Các file inputk.txt
|---Folder output:
	|--- Các file outputk.txt
|---File solveKB.py
|---File main.py

Cách chạy chương trình:
    B1: Đầu tiên cần vào folder output xóa tất cả các file outputk.txt đã chạy trước đó.
        (để chắc chắn rằng khi chạy các file inputk.txt mới cho kết quả chính xác)
    B2: Để các file inputk.txt vào thư mục input.
    B3: Mở file main.py và chạy chương trình.
    B4: Vào folder output và mở các file outputk.txt để xem kết quả.
	(Với file inputk.txt thì kết quả tương ứng là outputk.txt)
